Long live the Web

Long live the web explains what the web is and how it has evolved. The web is a powerful tool if used properly and the article explains how it is readily available for everyone to use. I liked how the author in the beginning suggests that the web might be "dead" in the future which made me wonder what could replace it if it does become obsolete. There are lot of security risks to data and privacy and the web is often an easy point for hackers to hack into. The article briefly mentions the risks but the web has more to offer. As mentioned in the article, making the web available for everyone might increase the literacy level as there is tons of information that could be found on the web. I wonder how the web would look like in 20 years and what more it has to offer.
